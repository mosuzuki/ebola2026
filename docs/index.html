Methods and source priority

Source priority

  1. National authorities: DRC Ministry of Health / INRB and Uganda Ministry of Health.
  2. WHO Disease Outbreak News, WHO AFRO, and official WHO statements.
  3. ECDC, Africa CDC, CDC, UKHSA, and other public health agencies.
  4. Operational partners such as MSF, IFRC, OCHA, Gavi, CEPI, and ReliefWeb.
  5. Peer-reviewed literature, preprints, and reputable science media.

Automation

The repository includes a GitHub Actions workflow and a Python update script. The workflow can be run daily and manually. The script preserves seed data and can be extended to parse official pages, RSS feeds, PDFs, or manually curated CSVs.

Recommended: require human review before overwriting official case counts because suspected and confirmed totals are frequently reclassified.
